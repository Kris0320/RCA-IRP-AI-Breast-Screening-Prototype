## What this snapshot contains
This snapshot contains a wider set of workflow diagrams and scenario notes, including existing radiologist read plus AI / machine read, concordance and adjudication paths, AI placement after the reader's initial judgement, and a CRDC or second-reading pathway. It also includes several case-level, lesion-level, and pathway-level disagreement notes. Some small sticky-note text is readable only partly, so precise transcription needs manual annotation.

## Why it matters to the IRP project
This image develops the workflow comparison into more granular disagreement scenarios. It appears to compare not only where AI sits in the workflow, but also what type of disagreement is produced: case-level discordance, lesion-level discordance, and pathway-level discordance. This supports the IRP's logic that AI placement creates different disagreement types, not a simple low-to-high risk scale.
