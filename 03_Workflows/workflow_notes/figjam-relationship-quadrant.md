## What this snapshot contains
This snapshot shows a relationship quadrant built from physical sticky notes and later digital notes. It maps human, AI, machine, and system relationships around trust, governance, peer discussion, audit, override, confidence scores, disagreement alerts, model versions, processed data, vendor updates, and performance monitoring. The axes appear to separate human intervention and machine / system dimensions, though the exact axis labels need manual confirmation.

## Why it matters to the IRP project
This snapshot supports the relationship-mapping side of the IRP. It shows that AI-human disagreement is not only a workflow event but a relationship problem between radiologists, AI outputs, institutional governance, data infrastructure, and technology vendors. It also gives design opportunities around transparency, override reasons, uncertainty communication, monitoring, and learning.

## Project use
This snapshot is useful for model development, reflection, and future prototype design. It gives direct material for the [[Relationship Quadrant]] note and can inform interface features such as override reason capture, AI confidence display, disagreement alerts, escalation guidance, and post-deployment monitoring.
