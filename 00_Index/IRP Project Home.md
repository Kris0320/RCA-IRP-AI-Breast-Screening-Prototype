# IRP Project Home

## Project direction
Designing a disagreement handling and learning system for AI-assisted breast screening.

## Current HMW
How might we help breast screening teams detect, resolve and learn from AI-human disagreements in a way that is accountable, transparent and consistent?

## Core focus
- AI-assisted breast screening
- AI-human reader disagreement
- Arbitration / senior review
- Accountability
- Transparency
- Consistency
- Audit trail
- Learning loop

## Current project logic
Tutorial 1 confirmed AI-human reader disagreement as a valid vertical focus.
Tutorial 2 unpacked relationship, trust calibration, traceability and accountability gaps.
Tutorial 3 pushed the project toward representative clinical workflow, decision hierarchy and scenario breakdown.
Tutorial 4 pushed the team to distinguish universal pattern mapping from a specific anchor model for intervention.
Tutorial 5 pushed the project to ground the anchor workflow inside organisational reality: actors, authority, information flow, arbitration protocol, people/process touchpoints and learning loop.


## Current decision
We use a universal disagreement pattern map for analysis, then anchor the intervention in AI-as-second-reader with human-led arbitration. The prototype should be a disagreement support service package, not only a dashboard.
