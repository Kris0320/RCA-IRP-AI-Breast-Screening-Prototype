# HMW and Design Direction

## Current HMW
How might we help breast screening teams detect, resolve and learn from AI-human disagreements in a way that is accountable, transparent and consistent?

## Project direction
Designing a disagreement handling and learning system for AI-assisted breast screening.

## Current strategy
Use a universal disagreement pattern map for analysis, then anchor the prototype in one specific model.

## Current anchor model
AI as second reader + human-led arbitration.
