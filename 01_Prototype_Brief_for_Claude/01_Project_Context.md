# Project Context

## Project title

AI-human disagreement handling for breast screening

## Project summary

This project explores how breast screening teams could detect, resolve and learn from disagreements between human readers and AI outputs. The prototype should demonstrate a case-based mammography reading moment where the human reader makes an independent judgement, AI output is then revealed, a disagreement is detected, and the system supports a non-blame handover, traceable arbitration and learning loop.

## Difference from DetectedX

DetectedX is mainly useful as a reference for case-based mammography reading, calibration and feedback. This prototype should not copy DetectedX directly. It should use a similar case-reading environment only as a familiar setting for testing an additional service layer: AI-human disagreement handling.

## Key design focus

- AI-human disagreement
- Arbitration support
- Decision traceability
- Non-blame handover
- Learning loop
